# Nexio – Modern Auth UI Template

Nexio is a modern, clean, and responsive authentication UI template built with HTML, CSS, and JavaScript.  
It includes Login and Register screens with smooth transitions, validation, and social login buttons.

## Features

- Modern and minimal design
- Login and Register forms
- Password strength validation
- Responsive layout (desktop & mobile)
- Social login buttons (Google, Facebook, LinkedIn, Twitter)
- Clean and well-structured code
- Easy to customize

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript

## File Structure

- `index.html` – Main template file
- `assets/css/style.css` – Styles
- `assets/js/script.js` – Form logic and validation
- `assets/image/` – Logo and social icons
- `docs/nexio-documentation.md` – Detailed usage guide
- `LICENSE.txt` – License file

## Usage

1. Open `index.html` in your browser to preview the template.
2. Customize colors, fonts, and layout in `assets/css/style.css`.
3. Adjust validation logic in `assets/js/script.js` if needed.
4. Integrate the template into your project or backend.
